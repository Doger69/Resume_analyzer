# Smart Resume Analyzer

##  Project Overview
Smart Resume Analyzer is a web-based application that analyzes resumes and matches them with job descriptions. It helps candidates understand how well their resume fits a specific job role and highlights missing skills.

##  Objectives
- Analyze resumes automatically
- Compare resume skills with job requirements
- Provide suggestions for improvement
- Help recruiters and candidates save time

##  Features
- Resume upload (PDF/DOC)
- Skill extraction using NLP
- Job description matching
- Resume score and feedback

##  Technologies Used
- Python
- Flask
- Natural Language Processing (NLP)
- HTML, CSS, JavaScript
- Machine Learning (if used)

##  How It Works
1. User uploads a resume
2. User enters or selects a job description
3. System extracts skills from resume
4. Resume is compared with job requirements
5. Feedback and match score are generated
